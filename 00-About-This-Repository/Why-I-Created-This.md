# Why I Created This Repository

## The Problem

While preparing for the AWS Certified Cloud Practitioner (CLF-C02) exam, I found that information was spread across multiple resources.

Important concepts were often scattered across:

* Documentation
* Videos
* Practice Tests
* Community Discussions

This made revision difficult.

---

## The Goal

I wanted a single place that contained:

* Exam-focused notes
* Domain-wise handbooks
* Service comparisons
* Exam traps
* Revision sheets

Everything organized in one repository.

---

## What Makes This Repository Different

Instead of only storing notes, this repository focuses on:

### Understanding Services

Learning what a service does and when to use it.

### Service Comparisons

Understanding differences between commonly confused services.

### Exam Traps

Highlighting mistakes frequently seen in practice tests.

### Fast Revision

Providing condensed notes for final-day review.

---

## Intended Audience

This repository is for:

* AWS beginners
* Students
* Developers
* Cloud enthusiasts
* CLF-C02 candidates

---

## Long-Term Vision

The repository started as a personal study resource.

The goal is to continuously improve it and help future learners prepare more efficiently for AWS certifications.

If this repository helps you, consider sharing it with others who are beginning their AWS journey.

Happy Learning and Best of Luck!
