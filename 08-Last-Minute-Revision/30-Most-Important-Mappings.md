# 30 Most Important AWS CLF-C02 Mappings

These are some of the most frequently tested service-to-purpose mappings in the AWS Certified Cloud Practitioner (CLF-C02) exam.

Memorize these before entering the exam.

---

## Security

Threat Detection → GuardDuty

Vulnerability Assessment → Inspector

Audit Logs → CloudTrail

Monitoring & Metrics → CloudWatch

Web Application Firewall → WAF

DDoS Protection → Shield

Identity & Access Management → IAM

Encryption Key Management → KMS

Sensitive Data Discovery → Macie

Security Dashboard → Security Hub

---

## Compute

Virtual Server → EC2

Serverless Compute → Lambda

Containers → ECS

Managed Kubernetes → EKS

Application Deployment Platform → Elastic Beanstalk

Automatic Scaling → Auto Scaling

Traffic Distribution → Elastic Load Balancer (ELB)

---

## Storage

Object Storage → S3

Block Storage → EBS

File Storage → EFS

Archive Storage → S3 Glacier

Hybrid Storage → Storage Gateway

Large-Scale Data Transfer → Snow Family

---

## Databases

Relational Database → RDS

High-Performance Relational Database → Aurora

NoSQL Database → DynamoDB

Data Warehouse → Redshift

Graph Database → Neptune

In-Memory Cache → ElastiCache

---

## Networking

DNS Service → Route 53

Content Delivery Network (CDN) → CloudFront

Private Network → VPC

Dedicated Private Connection → Direct Connect

Secure Encrypted Tunnel → VPN

API Management → API Gateway

---

## Messaging & Analytics

Notifications → SNS

Message Queue → SQS

Query Data In S3 Using SQL → Athena

Streaming Data Processing → Kinesis

Business Intelligence Dashboards → QuickSight

---
