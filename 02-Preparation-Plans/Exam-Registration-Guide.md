# AWS CLF-C02 Exam Registration Guide

This guide explains how to create your AWS Certification account, schedule the AWS Certified Cloud Practitioner (CLF-C02) exam, and prepare the required documents before exam day.

---

# Before You Register

Before booking the exam, it is recommended to complete the official AWS Cloud Practitioner Essentials course.

Official AWS Skill Builder Course:

https://skillbuilder.aws/learn/94T2BEN85A/aws-cloud-practitioner-essentials/8D79F3AVR7

This free course covers most of the concepts required for the AWS Certified Cloud Practitioner (CLF-C02) exam and is one of the best starting points for beginners.

# Step 1 — Create an AWS Certification Account

Visit:

https://www.aws.training

Sign in using:

* Existing AWS account
* Amazon account
* Create a new account

Complete:

* Full Name
* Email Address
* Country
* Time Zone

Make sure your name exactly matches the identification document you will use on exam day.

---

# Step 2 — Open AWS Certification Dashboard

After signing in:

1. Open AWS Certification Dashboard
2. Navigate to:

Certification → Schedule New Exam

3. Select:

AWS Certified Cloud Practitioner (CLF-C02)

---

# Step 3 — Choose Exam Delivery Method

AWS provides two options:

## Option 1 — Pearson VUE Test Center

Advantages:

* Stable environment
* No home internet issues
* Professional testing center

Recommended for most candidates.

---

## Option 2 — Pearson VUE Online Proctored Exam

Advantages:

* Write from home
* Flexible scheduling

Requirements:

* Webcam
* Microphone
* Quiet room
* Stable internet connection
* Clean desk

---

# Step 4 — Schedule The Exam

Choose:

* Country
* Time Zone
* Exam Date
* Exam Time

Review the details carefully before proceeding.

---

# Step 5 — Complete Payment

Proceed to checkout and complete payment.

If you have:

* AWS discount voucher
* Promotional code
* Certification discount

Apply it before payment.

After successful payment:

* Exam confirmation email will be sent
* Calendar invitation may be generated
* Appointment appears in AWS Certification Dashboard

---

# Required Identification Documents

Bring a valid government-issued identification document.

Commonly accepted documents:

### Aadhaar Card

### PAN Card

### PVC Aadhaar Card

The name on your identification document should match the name used during registration.

Always verify the latest Pearson VUE identification requirements before the exam.

---

# Exam Day Checklist

Before the exam:

* Verify exam date and time
* Keep identification document ready
* Join early
* Ensure stable internet connection (online exam)
* Keep desk clean
* Close unnecessary applications

---

# Exam Overview

Exam:

AWS Certified Cloud Practitioner (CLF-C02)

Format:

* Multiple Choice Questions
* Multiple Response Questions

Duration:

90 Minutes

Language:

Multiple languages available

Passing Score:

AWS does not publicly disclose an exact passing percentage.

Scores are reported on a scale.

Do not wait until the last day to create your certification account.

Complete account setup, identification verification, and exam scheduling early so that your final days can be focused entirely on preparation and revision.
