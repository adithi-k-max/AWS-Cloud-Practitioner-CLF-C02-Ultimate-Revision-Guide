# Learning Philosophy

Understanding concepts is more important than memorizing answers.

Focus on:

- Service Use Cases
- Service Comparisons
- Security Concepts
- AWS Keywords
- Elimination Techniques

The exam becomes significantly easier when you understand when and why a service should be used.
